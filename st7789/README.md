# W55MH32 ST7789 SPI display

In this example, it used a 1.54 inch, 240 * 240 ST7789 SPI display.

This example scripted used russhughes's st7789pyrepo and library.
The mpy file has converted for W55MH32 to use. 

For more information about the st7789py repo and library, please refer to this [link](https://github.com/russhughes/st7789py_mpy/tree/master)
